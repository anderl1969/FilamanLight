About FilaMan Lite

FilaMan Lite is designed as add-on to FilaMan. It allows you to read and write NFC tags, which are used by FilaMan to automatically sync data with the self-hosted Spoolman platform.

FilaMan Lite connects to Spoolman for reading spool and location data, before writing it to a NFC tag.

FilaMan Lite does NOT write any information to Spoolman!

Get more information at https://www.filaman.app and https://github.com/ManuelW77/Filaman.

Writing Spool Tags

Writes spool data from Spoolman to a NFC tag.

Writing Location Tags

Writes location data from Spoolman to a NFC tag.

Reading Tags

Reads FilaMan compatible NFC tags and displays the stored data.
